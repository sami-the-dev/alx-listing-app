## ALX Listing App

ALX Listing App is a web application designed to replicate the core functionality of an Airbnb listing page. The project aims to provide users with a seamless experience for browsing, filtering, and viewing property listings.

### Project Structure

- **components/**: Contains reusable React components used throughout the app, such as listing cards, filters, and navigation elements.
- **interfaces/**: Defines TypeScript interfaces and types for data models, ensuring type safety across the project.
- **constants/**: Stores static values and configuration options, such as API endpoints and default settings.
- **public/assets/**: Includes static files like images, icons, and other assets accessible by the application.

### Getting Started

1. **Install dependencies**  
   Run the following command in the project root:

   ```bash
   npm install
   ```

2. **Start the development server**  
   Launch the app locally with:

   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000` to view the application.
